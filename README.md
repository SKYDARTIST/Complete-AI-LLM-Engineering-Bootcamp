# Complete AI & LLM Engineering Bootcamp

My personal code repository for the **Full Stack Generative and Agentic AI with Python** course by [Hitesh Choudhary](https://www.udemy.com/user/hiteshchoudharyofficial/) and [Piyush Garg](https://www.udemy.com/user/piyush-garg-6/) on Udemy.

---

## What This Course Covers

- Python from scratch — syntax, OOP, advanced features
- Git, GitHub, Docker, Pydantic
- How LLMs work — tokenization, embeddings, attention, transformers
- Prompt engineering — zero-shot, few-shot, chain-of-thought
- OpenAI & Gemini API integration
- Running models locally with Ollama + Docker
- RAG pipelines with LangChain and vector databases
- LangGraph — stateful AI with nodes, edges, checkpointing
- Memory systems — short-term, long-term, episodic, semantic (Mem0, Neo4j)
- AI voice agents — STT + LLM + TTS
- Model Context Protocol (MCP) — build MCP servers with Python
- Multi-modal AI — process images + text together
